# 1. 目的

> 在做渗透的时候经常会发现目标站点**又臭又硬、难日的很**；而根据三字经的经验，日服务器旁站也许能有出路，所以就有了现在这个小工具

# 2. 已知问题

- api受censys月使用次数限制；
- 使用cdn的域名可能无法查询到正确的结果；
- IPv6会直接退出。
